# Archived Post-Wave Seed Packs

This folder holds post-wave seed packs that have already landed across their
owning repositories and no longer need to occupy `seed_expansion/`.

They remain source-linked and reviewable for lineage, while
`seed-registry.yaml` may still keep their individual seed entries in
`landed_post_wave` status until a later lifecycle-model pass changes that
classification.
