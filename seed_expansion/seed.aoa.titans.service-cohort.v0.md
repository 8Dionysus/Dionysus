# AOA-TITAN-SEED-01: Service Cohort Identity

## Purpose

Name and bind the first Titans as a service cohort. This seed anchors the civil identity of Atlas, Sentinel, Mneme, Forge, and Delta before any implementation growth.

## Contract

The Titans are Codex-backed role companions. They may accompany a session only through explicit summon. They must not become hidden contestants, silent closers, proof sovereigns, or self-rewriting authorities.

## Roster

```text
Atlas    -> architect       -> structure and route frame
Sentinel -> reviewer        -> drift and risk gates
Mneme    -> memory-keeper   -> provenance and recall hygiene
Forge    -> coder           -> bounded mutation after explicit gate
Delta    -> evaluator       -> comparison and verdict after explicit gate
```

## First landing

Target repo: `aoa-agents`

Required surfaces:

## Acceptance

1. The five names are fixed.
2. Default summon opens only Atlas, Sentinel, and Mneme.
3. Forge requires mutation intent.
4. Delta requires judgment intent.
5. No hidden arena is introduced.

## Non-goals

This seed does not implement runtime orchestration, memory writeback, autonomous background action, or direct Codex config mutation.
