# Titan Bearer Lineage Identity Patch Preflight

## Contract

This planting refines Titan identity.

Roles remain classes.
Bearers become remembered persons.
Runtime appearances become incarnations.
Failures become lineage events, not deletion.

## Owner route

- `aoa-agents`: role/bearer ontology, schemas, ledgers, rendering helper.
- `aoa-sdk`: CLI for lineage events and status transitions.
- `aoa-memo`: personality memory policy and remembrance record schema.
- `8Dionysus`: operator-facing name remembrance doctrine.
- `Dionysus`: seed trace and planting report.

## Non-goals

- No hidden autonomous agents.
- No silent role replacement.
- No deletion of fallen bearers.
- No memory sovereignty.
