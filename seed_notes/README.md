# Seed Notes

`seed_notes/` holds informal seed texts that are worth preserving but should
not be mistaken for canonical live control surfaces.

Use these notes for:

- exploratory thinking
- donor scouting
- early seed extraction
- historical idea lineage

Do not treat this folder as:

- the live queue
- owner-repo truth
- a second registry

See `exploratory/README.md` for the current exploratory note family.
