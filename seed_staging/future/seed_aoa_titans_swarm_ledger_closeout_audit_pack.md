# AoA Titans Swarm Ledger, Closeout Audit & Summon Discipline Pack

## Purpose

The Fifteenth Wave gave Titan names a session spine.
The Sixteenth Wave gives Titan participation a durable swarm ledger.

```text
incarnation
  -> task contract
    -> agent report
      -> finding lifecycle
        -> grade and timeout accounting
          -> closeout audit
            -> memory candidate
```

## Core lessons from field sessions

- Atlas and Mneme should be summoned early.
- Sentinel should receive literal diff excerpts and one narrow review slice.
- Delta should receive one bounded claim, evidence bundle and residual-risk question.
- Low grades must distinguish Titan fault from orchestration fault.
- Compaction-recovered session material is usable, but must carry chronology confidence limits.

## Non-goals

- No hidden autospawn.
- No agent report becomes memory by itself.
- No Delta certification sovereignty.
- No Forge mutation without a Fifteenth-Wave mutation gate.
- No generic role shadowing.
