# seed.tos.pre-expansion-soil.v0.md

Roadmap-derived seed source for pre-expansion ToS soil work before manual Zarathustra entry.

Date: 2026-03-22
Status: curated roadmap seed, not direct implementation backlog

## Grounding

The first bounded lineage pilot already exists.

The next move should not yet become wider world-thought expansion.
It should prepare the soil for later growth while tree motion stays explicitly gated:

- first context-node template
- explicit manual corpus-entry gate
- pre-expansion review route

This keeps ToS structurally ready without pretending plurality has already been earned through new branch growth.

---

<a id="tos-soil-context-node-template"></a>
## Context-Node Template

### Core move

Give ToS its first public context-node template so temporal, geographic, linguistic, civilizational, and institutional context can have a disciplined authored form.

### Why now

- `context` is already part of the public node-family model
- context law can be prepared before any new corpus slice is opened
- this strengthens future plurality without forcing premature branch growth

### First materialized output

- context-node template note
- additive node-contract support for context-facing relation labels

---

<a id="tos-soil-manual-zarathustra-entry-gate"></a>
## Manual Zarathustra Entry Gate

### Core move

Make explicit that tree motion does not open automatically.

### Why now

- the next real movement in the tree should begin from maintainer-led source entry
- manual Zarathustra work is the present gate, not an implicit future detail
- this prevents empty template work from pretending to be real corpus growth

### First materialized output

- manual corpus-entry gate note
- explicit statement that wider expansion remains later until manual source entry begins

---

<a id="tos-soil-pre-expansion-review-route"></a>
## Pre-Expansion Review Route

### Core move

Harden the review route for preparation work that is allowed before wider expansion begins.

### Why now

- ToS needs a way to prepare soil without silently opening new families or branch nodes
- review law should say what counts as preparation and what counts as actual tree movement
- this keeps the current pause legible to future contributors and coding agents

### First materialized output

- pre-expansion soil doctrine note
- review checklist updates
- review note for the soil-prep wave

---

<a id="tos-soil-wider-world-thought-expansion-later"></a>
## Held Later: Wider World-Thought Expansion

### Why later

- plurality should not be imitated before new source entry begins
- wider expansion should follow real corpus movement rather than precede it
- the current gate remains manual Zarathustra entry
